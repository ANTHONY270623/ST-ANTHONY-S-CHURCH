## 🔍 **TROUBLESHOOTING:**

### **If Download Doesn't Work:**
1. Check if `baptism-form.pdf` exists in project folder
2. Try right-click → "Save As" on download button
3. Check browser settings allow downloads

### **If Print Doesn't Work:**
1. Enable pop-ups for your website
2. Try the View button then Ctrl+P
3. Download PDF and print manually

### **If Buttons Look Wrong:**
1. Check if `form-actions-styling.css` is loading
2. Clear browser cache and refresh
3. Test in different browser
