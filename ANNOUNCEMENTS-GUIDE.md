## 🔧 **Why PHP Files Download Instead of Running**

**Problem**: When you click `.php` files, they download instead of executing.

**Solution**: You need a web server with PHP. Here are your options:

### **Option 1: XAMPP (Recommended)**
1. Download XAMPP from https://www.apachefriends.org/
2. Install and start Apache + MySQL
3. Copy your project to `C:\xampp\htdocs\church`
4. Access via `http://localhost/church`

### **Option 2: WAMP**
1. Download WAMP from http://www.wampserver.com/
2. Install and start services
3. Copy project to `C:\wamp64\www\church`
4. Access via `http://localhost/church`

### **Option 3: Use the Static Version (Current)**
- No server needed
- Works by double-clicking HTML files
- Uses browser storage instead of database
- Perfect for testing and small usage
